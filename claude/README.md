# Claude Documentation

このフォルダはClaude Code用のドキュメントを管理するためのものです。

## 構造

- `/architecture/` - システム設計とアーキテクチャ
- `/api/` - APIドキュメント
- `/commands/` - コマンドの仕様と実装詳細
- `/events/` - イベントハンドラーの詳細
- `/models/` - データモデルの定義
- `/workflows/` - 開発ワークフローと手順

## ルール

- 各ファイルは300行以内
- 詳細な情報は適切なサブディレクトリに分割
- PlantUMLファイル（.puml）も活用
